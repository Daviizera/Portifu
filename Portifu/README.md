# Atividade-01 de Desenvolvimento Web II.

## Descrição:
Portifu é um projeto simples de portfólio ficticio. Ele contém uma página inicial com links de navegação para diferentes seções, incluindo "Projetos" e "Contato". O objetivo principal para mim foi demonstrar habilidades básicas em HTML e CSS.

## Estrutura do Projeto:
- **index.html**: Contém a estrutura principal da página.
- **styles.css**: Define o estilo visual da página.
- **img/**: Diretório para armazenar imagens utilizadas no projeto.

## Funcionalidades:
- Navegação entre seções da página.
- Formulário de contato funcional.

## Alterações Recentes:
- Corrigido o botão "Contato" para redirecionar corretamente para a seção de formulário.

## Como Visualizar:
1. Abra o arquivo `index.html` em qualquer navegador.
2. Navegue pelas seções utilizando os links no menu.

## Tecnologias Utilizadas:
- HTML5
- CSS3
